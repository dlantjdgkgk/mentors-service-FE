# mentors 프로젝트

## Introduce
멘토링 서비스 컨셉의 커뮤니티

## Installation
```
1. git clone https://github.com/mentors-service/mentors-service-FE.git

2. cd mentors-service-FE

3. npm install

4. npm run start
```

## Setting
```
# .env.development.local

REACT_APP_BASE_URL=백엔드 API 주소
REACT_APP_KAKAO_OAUTH_URL=백엔드 카카오 OAuth 주소
```
